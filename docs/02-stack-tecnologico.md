## Stack Tecnológico 
- **Framework** [Next.js 15+] (App Router)
- **Frontend** [React 19] 
- **Componentes UI** [Shadcn UI (New York), Tailwind CSS v4 (OKLCH)]
- **Lenguaje** [TypeScript 5]
- **Backend / DB** [Supabase] (PostgreSQL + RLS)
- **Gestión de estado** [Zustand 5]
- **Validación** Zod 4
- **Iconos** Lucide React
- **Carrusel** Embla Carousel
