# Guía para el proyecto amvill 

Este documento define las directrices y estandares para el desarrollo del proyecto **amvill**, será un Catálogo Digital (sin pasarela de pago por ahora) para la "venta" de Perfumes Decant.

## General
- Utliza buenas prácticas para el desarrollo de un SaaS (Software as a Service)
- Estructura de proyecto escalable (por ejemplo en un futuro implementar la pasarela de pago)
- La app soporta español e ingles, siendo español su idioma principal
